Adding content!

Everything we've done so far is about defining the shape of our webpage. Let's looks at adding content! Here are some of the most common content tags to get you started.

<h1>…<h6> = Headings. h1 is the top-level heading. The text gets smaller as we go higher – later we'll look at how we can style these.
<p> = A paragraph.
<br /> = A forced line break.
<a> = A link (“a” stands for “anchor”).
<img> = An embedded image.
<ul>, <ol> = Unordered (bullet points) and ordered (numbered) lists.
<li> = List item. An item in a list.
<table> = A table, which is made up of rows (<tr>), which are made up of header cells (<th>) and data cells (<td>).
