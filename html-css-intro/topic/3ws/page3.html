How do we make a website?

Two Main Parts

  • HTML = Hyper-Text Markup Language.
    This is our content. We define the text, images, videos, etc, that we want on our page, using HTML.
  • CSS = Cascading Style Sheets.
    This is our style. We can define color, font, and layout.

Quick Quiz

Look at different parts of a website, and think about whether they are HTML (content) or CSS (style).
