Set up your tools

  1. Check that VS Code and GutHub Desktop are installed.
  2. Create a folder called GitHub under your C:\Users\<your-user-name>\ folder.

A template to get you started

  1. Clone the website template from GIT:
    web-design-project-1 ← When asked to Open GitHubDesktop.exe, click Open.
  2. Select the C:\Users\<your-user-name>\GitHub\ folder, and change web-design-project-1 to my website (or something similar).
  3. The cloned site will appear in GitHub Desktop. Click the Open in Visual Studio Code button.
  4. Open the README.md file and then click the preview button.
  5. Follow the instructions to set up your project.
