Mix & Match Selectors

CSS selectors apply a style based on:

tag : Examples: body, main, nav, h1, p,
id : The id attribute on an HTML element. Id's are indicated by a hash (#).
Example: #logo would refer to an element with id="logo"
class : The class attribute on an HTML element. Id's are indicated by a hash (#).
Example: .highlight would refer to an element with class="highlight"

Examples:

p { color: red; } All paragraphs will have red text.
#logo { width: 200px; } The logo will be 200 pixels wide.
.highlight { background: yellow; } Everything with class="highlight" will have a yellow background.
Common CSS properties and values: color – The colour of the text. font-family – A list of names of fonts (with fallback). margin – Leaves space around the element. padding – Leaves space between the element’s content and it’s border. background – Sets a background colour, gradient, or image. border – Set the border.
