Mix & Match Selectors

CSS selectors apply a style based on:

tag : Examples: body, main, nav, h1, p,
id : The id attribute on an HTML element. Id is indicated by a hash (#).
Example: #logo would refer to an element with id="logo"
class : The class attribute on an HTML element. Class is indicated by a dot (.).
Example: .highlight would refer to elements with class="highlight"

Examples:

p { color: red; } All paragraphs will have red text.
#logo { width: 200px; } The logo will be 200 pixels wide.
.highlight { background: yellow; } Everything with class="highlight" will have a yellow background.
