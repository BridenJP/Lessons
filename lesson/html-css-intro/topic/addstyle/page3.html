Common CSS properties and values include:

color : The colour of the text.
font-family : A list of names of fonts (with fallback).
margin : Leaves space around the element.
padding : Leaves space between the element’s content and it’s border.
background : Set a background colour, gradient, or image.
border : Set the border.
