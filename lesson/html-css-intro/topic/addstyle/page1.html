Stylin'

Once we have some content, it's time to make it look good. Let's give it some style!

The way we do that is with CSS (Cascading Style-Sheets).

Where to we add style?

We can add style directly into our HTML document using the <style> tag, and we can also put style directly on many HTML tags, but it's considered "best practice" to put them in a separate file. We'll come back to this.

CSS selectors

The CSS puts style onto one or more parts of our HTML.

We identify which parts with a selector.
