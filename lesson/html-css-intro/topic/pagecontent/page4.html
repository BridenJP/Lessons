Where does it all go?

Inside the <body> of the document, we can start organising our content. HTML has some specific tags for common parts of a web-page.

<header> = A header for the top of the page.
<nav> = A navigation bar - a menu that lets us go to different parts of the website.
<main> = The main content area.
<section> = A tag to define a specific section in the content.
<footer> = A footer for the bottom of the page.
<div> = A general purpose tag for structuring content (div = division).
