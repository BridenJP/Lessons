Simple Format Tags

<b>some text</b> = bold.
<i>some text</i> = italic.
<b><i>some text</i></b> = bold and italic.

Most tags come in pairs. They have a begin tag and an end tag. The end tag includes the slash.

If you start with <b> and don't include the </b> at the end, the browser will just keep bolding EVERTHING!
