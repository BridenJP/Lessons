The document

The webpage is referred to as an HTML "document". As well as having content the browser gets other important information from the document.

Document tags

The document uses tags to define its overall structure.

<!doctype html> = tells the browser we are using the latest HTML.
<html>…</html> = the entire HTML document.
<head>…</head> = the document header includes information about the HTML page.
<body>…</body> = the actual content of the page.

Notice how these tags are also generally in pairs.
